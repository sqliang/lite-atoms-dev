# Project Summary
AI Workspace is an innovative developer tool designed to enhance productivity by facilitating seamless interactions with AI through a chat interface, integrated code editor, and a preview workspace. Users can engage with AI using natural language while managing diverse content types, including code, images, and documents. The application features a user-friendly design inspired by Notion, with a structured file tree for easy navigation. Recent enhancements include improved project name handling, automatic insertion of mock project data into Supabase, and an updated user experience through refined UI components.

# Project Module Description
The project consists of the following functional modules:
- **Chat Panel**: Users interact with AI, featuring message history, attachment previews, and improved navigation tools.
- **Workspace Panel**: A multi-tabbed editor for various content types, allowing users to manage open files effectively.
- **Homepage**: Users can describe projects and view recent projects, with user status information prominently displayed.
- **Context Management**: Manages active tabs and workspace states, including project files.
- **Authentication**: Integrated Supabase for user authentication, including sign-in, sign-up, and session management.
- **Mock Data Management**: Automatically populates mock projects in Supabase for new users, with improved logic to handle existing projects.

# Technology Stack
- **React 18**: Frontend library for building UI components.
- **TypeScript**: Superset of JavaScript for type safety.
- **Vite**: Build tool for fast development and production builds.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **shadcn/ui**: Component library for UI elements.
- **react-resizable-panels**: Library for creating resizable panels.
- **highlight.js**: Syntax highlighting library for code display.
- **JSZip**: Library for generating ZIP files.
- **file-saver**: Library for saving files on the client-side.
- **Supabase**: Backend as a service for authentication and database management.

# Usage
To get started with the project:
1. Install dependencies:
   ```bash
   cd app/frontend
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run the application:
   ```bash
   pnpm run lint
   ```
