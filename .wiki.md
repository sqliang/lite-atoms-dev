# Project Summary
AI Workspace is an innovative developer tool designed to enhance productivity by facilitating seamless interactions with AI through a chat interface, integrated code editor, and preview workspace. Users can communicate with AI using natural language while editing and previewing various content types, including code, images, and documents. The application features a Notion-inspired design for improved usability, with a file tree structure for easy navigation. Recent updates include the integration of Supabase for user authentication, allowing users to register and log in securely. A user avatar and logout functionality have also been added to enhance user experience.

# Project Module Description
The project consists of the following functional modules:
- **Chat Panel**: Users interact with AI, featuring message history, attachment previews, enhanced message alignment, and a scroll-to-bottom button. The input area has been restructured with a full-width textarea on top and a toolbar row below for the send button. A back-to-home button has been added to facilitate navigation.
- **Workspace Panel**: A multi-tabbed editor for various content types, with horizontal scrolling for tabs and a "close all" button for managing open files.
- **Homepage**: Allows users to describe projects and view recent projects. The project creation button has been removed, and session IDs are now utilized for navigation.
- **Context Management**: Manages active tabs and workspace states, including project files.
- **Authentication**: Integrated Supabase for user authentication, including sign-in, sign-up, and session management.

# File Description Inventory

- **index.html**: Entry point for the web application.
- **package.json**: Lists dependencies and scripts for building and running the project.
- **postcss.config.js**: Configuration for processing CSS.
- **prerender/**: Scripts for pre-rendering blog routes and generating sitemaps.
- **public/**: Contains static assets like images and favicons.

# Technology Stack
- **React 18**: Frontend library for building UI components.
- **TypeScript**: Superset of JavaScript for type safety.
- **Vite**: Build tool for fast development and production builds.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **shadcn/ui**: Component library for UI elements.
- **react-resizable-panels**: Library for creating resizable panels.
- **highlight.js**: Syntax highlighting library for code display.
- **JSZip**: Library for generating ZIP files.
- **file-saver**: Library for saving files on the client-side.
- **Supabase**: Backend as a service for authentication and database management.

# Usage
To get started with the project:
1. Install dependencies:
   ```bash
   cd app/frontend
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run the application:
   ```bash
   pnpm run lint
   ```
