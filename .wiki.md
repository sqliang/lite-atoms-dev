# Project Summary
AI Workspace is a modern developer tool designed to enhance interactions with AI through a chat interface, integrated editor, and preview workspace. This application allows users to engage with AI using natural language while seamlessly editing and previewing various content types, including code, images, and documents. It features a Notion-inspired theme for a user-friendly interface and introduces a file tree structure for easy navigation of project files. Recent updates have added functionalities for file copying, downloading individual files, and downloading entire project files in a ZIP format.

# Project Module Description
The project consists of the following functional modules:
- **Chat Panel**: An interactive area for users to communicate with the AI, featuring message history and attachment previews.
- **Workspace Panel**: A multi-tabbed editor and viewer for code, images, and documents, enhanced with a file tree for easy navigation and new toolbar actions for file management, including file copying and downloading.
- **Context Management**: A shared state management system for handling active tabs and workspace states, including project files.

# Technology Stack
- **React 18**: Frontend library for building UI components.
- **TypeScript**: Superset of JavaScript for type safety.
- **Vite**: Build tool for fast development and production builds.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **shadcn/ui**: Component library for UI elements.
- **react-resizable-panels**: Library for creating resizable panels.
- **highlight.js**: Syntax highlighting library for code display.
- **JSZip**: Library for generating ZIP files.
- **file-saver**: Library for saving files on the client-side.

# Usage
To get started with the project:
1. Install dependencies:
   ```bash
   cd app/frontend
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run the application:
   ```bash
   pnpm run lint
   ```
