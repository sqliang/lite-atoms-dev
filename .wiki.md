# Project Summary
AI Workspace is a developer tool designed to facilitate interaction with AI through a chat interface while providing an integrated editor and preview workspace. It features a dark mode and an IDE-like interface, allowing users to engage with AI using natural language and seamlessly edit and preview various content types such as code, images, and documents.

# Project Module Description
The project consists of the following functional modules:
- **Chat Panel**: An interactive area for users to communicate with the AI, featuring message history and attachment previews.
- **Workspace Panel**: A multi-tabbed editor and viewer for code, images, and documents, enabling users to edit and view content side by side.
- **Context Management**: A shared state management system for handling active tabs and workspace states.

# Technology Stack
- **React 18**: Frontend library for building UI components.
- **TypeScript**: Superset of JavaScript for type safety.
- **Vite**: Build tool for fast development and production builds.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **shadcn/ui**: Component library for UI elements.
- **react-resizable-panels**: Library for creating resizable panels.
- **highlight.js**: Syntax highlighting library for code display.

# Usage
To get started with the project:
1. Install dependencies:
   ```bash
   cd app/frontend
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run the application:
   ```bash
   pnpm run lint
   ```
