# Project Summary
AI Workspace is a modern developer tool that enhances interaction with AI through a chat interface, integrated editor, and preview workspace. It allows users to utilize natural language to engage with AI while seamlessly editing and previewing diverse content types, including code, images, and documents. The application now features a Notion-inspired theme, providing a user-friendly interface that aligns with popular productivity tools. Recently, a file tree structure has been integrated, allowing users to navigate project files easily.

# Project Module Description
The project consists of the following functional modules:
- **Chat Panel**: An interactive area for users to communicate with the AI, featuring message history and attachment previews.
- **Workspace Panel**: A multi-tabbed editor and viewer for code, images, and documents, now enhanced with a file tree for easy navigation.
- **Context Management**: A shared state management system for handling active tabs and workspace states, including project files.

# Technology Stack
- **React 18**: Frontend library for building UI components.
- **TypeScript**: Superset of JavaScript for type safety.
- **Vite**: Build tool for fast development and production builds.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **shadcn/ui**: Component library for UI elements.
- **react-resizable-panels**: Library for creating resizable panels.
- **highlight.js**: Syntax highlighting library for code display.

# Usage
To get started with the project:
1. Install dependencies:
   ```bash
   cd app/frontend
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run the application:
   ```bash
   pnpm run lint
   ```
