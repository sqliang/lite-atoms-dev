# Project Summary
Lite Atoms Dev is an innovative developer tool aimed at enhancing productivity through seamless interactions with AI. It features a chat interface, integrated code editor, and a preview workspace, allowing users to manage various content types such as code, images, and documents using natural language. The application adopts a user-friendly design inspired by Notion, with a structured file tree for easy navigation. Recent updates include improved project name handling, automatic insertion of mock project data into Supabase, and a refined user experience with enhanced UI components.

# Project Module Description
The project consists of the following functional modules:
- **Chat Panel**: Facilitates user interactions with AI, featuring message history, attachment previews, and improved navigation tools.
- **Workspace Panel**: A multi-tabbed editor for managing various content types and open files effectively.
- **Homepage**: Displays project descriptions and recent projects, with user status information prominently featured.
- **Context Management**: Manages active tabs and workspace states, including project files.
- **Authentication**: Integrated Supabase for user authentication, encompassing sign-in, sign-up, and session management.
- **Mock Data Management**: Automatically populates mock projects in Supabase for new users, with improved logic for handling existing projects.
- **Code Editor**: Enhanced with support for code folding, allowing users to collapse or expand code blocks based on braces/brackets.

- **shadcn/ui**: Component library for UI elements.
- **react-resizable-panels**: Library for creating resizable panels.
- **highlight.js**: Syntax highlighting library for code display.
- **JSZip**: Library for generating ZIP files.
- **file-saver**: Library for saving files on the client-side.
- **Supabase**: Backend as a service for authentication and database management.

# Usage
To get started with the project:
1. Install dependencies:
   ```bash
   cd app/frontend
   pnpm install
   ```
2. Configure environment variables:
   Create a `.env.local` file in the project root and add your Supabase configuration:
   ```env
   VITE_SUPABASE_URL=your_supabase_project_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```
3. Start the development server:
   ```bash
   pnpm run dev
   ```
4. Build the project:
   ```bash
   pnpm run build
   ```
