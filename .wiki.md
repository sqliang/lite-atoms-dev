# Project Summary
AI Workspace is an advanced developer tool that enhances productivity by enabling seamless interactions with AI through a chat interface, integrated code editor, and preview workspace. Users can communicate with AI using natural language while editing and previewing various content types, such as code, images, and documents. The application features a Notion-inspired design for improved usability, including a file tree structure for easy navigation. Recent enhancements include optimized input areas in the chat panel, improved chat functionality, better handling of multi-line text, and accurate file operation cards that now correctly open the actual content of attachments.

# Project Module Description
The project consists of the following functional modules:
- **Chat Panel**: Users interact with AI, featuring message history, attachment previews, enhanced message alignment, and a scroll-to-bottom button. The input area has been optimized to remove unnecessary disclaimer text and now features an auto-expanding textarea.
- **Workspace Panel**: A multi-tabbed editor for various content types, with horizontal scrolling for tabs and a "close all" button for managing open files.
- **Homepage**: Allows users to describe projects and view recent projects. The project creation button has been removed, and session IDs are now utilized for navigation.
- **Context Management**: Manages active tabs and workspace states, including project files.

# Technology Stack
- **React 18**: Frontend library for building UI components.

- **Vite**: Build tool for fast development and production builds.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **shadcn/ui**: Component library for UI elements.
- **react-resizable-panels**: Library for creating resizable panels.
- **highlight.js**: Syntax highlighting library for code display.
- **JSZip**: Library for generating ZIP files.
- **file-saver**: Library for saving files on the client-side.

# Usage
To get started with the project:
1. Install dependencies:
   ```bash
   cd app/frontend
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run the application:
   ```bash
   pnpm run lint
   ```
